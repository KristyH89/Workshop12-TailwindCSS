# 🌿 Kristy's Cooking Adventures – Tailwind CSS Landing Page

This repository contains my **Tailwind CSS Landing Page**, created for the *Lexicon Tailwind Workshop*.  
The goal of the assignment was to design a modern, responsive landing page using Tailwind utility classes.

The landing page serves as a **promotional entry point** to my main website:  
**Kristy's Cooking Adventures** — a cozy place full of recipes, inspiration, and everyday cooking ideas.

---

## 📘 Workshop Instructions

👉 [View workshop instructions](TailwindCSS-Workshop.md) 

---

## 🎯 Project Goals

- Build a **modern landing page** using Tailwind CSS  
- Use **utility‑first classes** for layout, spacing, colors, and typography  
- Create a **fully responsive** design  
- Apply a **clean and consistent visual style**  
- Include a clear **call‑to‑action**  
- Design something **creative and personal**

---

## 🌟 Features

- ✔ Clickable **logo banner**  
- ✔ Modern **hero section** with image  
- ✔ “Explore Recipes” button styled in my brand colors  
- ✔ Three clickable category cards:
  - Easy Dinners  
  - World Flavors  
  - Sweet Treats  
- ✔ Soft green **CTA banner**  
- ✔ Clean and simple **footer**  
- ✔ Fully responsive layout  
- ✔ Custom Tailwind colors using arbitrary values
